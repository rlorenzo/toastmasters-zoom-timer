Setup Guide

🎦 OBS Studio Virtual Camera Recommended

OBS captures this browser tab and exposes it as a system-level virtual camera that Zoom can select natively — the audience sees only the composite, no browser chrome.

  1. Install OBS Studio from obsproject.com (free, Windows / macOS / Linux).
  2. In OBS, click + under Sources and choose Window Capture (Windows) or macOS Screen Capture.
  3. Select this browser window or tab from the source dropdown. Resize the source to fill the scene at 1920×1080.
  4. Click Start Virtual Camera in the OBS control panel (bottom-right).
  5. In Zoom → Settings → Video, select OBS Virtual Camera as your camera.
  6. Press H here to enter Stage Mode (hides controls) so OBS captures a clean feed.

📷 Zoom Share Window

No extra software needed — share this browser tab directly in Zoom. The audience sees the canvas; you see the controls in your presenter view.

  1. In Zoom, click Share Screen.
  2. Switch to the Window or Advanced tab and select this browser window.
  3. Enable "Optimize for video clip" for smoother frame rate.
  4. Press H here to enter Stage Mode so viewers see only the composited canvas — no sidebar, no controls.
  5. Start the timer via keyboard shortcut (Space) so you don't need to click back to this window.

Note: Zoom's share-window path captures your whole browser tab, including the page background. Stage Mode minimises visible chrome to just the canvas.
